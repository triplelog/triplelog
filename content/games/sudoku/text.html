Rules of Sudoku

  • Each row must contain each number from 1-9 exactly once.
  • Each column must contain each number from 1-9 exactly once.
  • Each 3x3 block must contain each number from 1-9 exactly once.

Basic Strategy

  • Look for numbers that must be placed in a particular cell. Video Scenario
  • Look for cells that only have one acceptable option. Video Scenario
  • Find a sub-row or sub-column that must contain a certain number. Video Scenario
