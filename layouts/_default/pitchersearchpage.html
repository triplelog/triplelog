Search by name or MLB ID. Or click the dropdown button to browse pitchers.

What do the pitch charts show?

  • Each chart displays the pitch sequence in an at-bat.
  • Balls are faded while strikes are not.
  • Swings are represented by a border around the ball.
  • Each pitch type has its own color.
  • Fastballs are red
  • Curveballs are green
  • Sliders are yellow
  • Changeups are blue
  • Others are black
The pitch type information is the default data provided by MLBAM, which is good for most pitchers. However, some pitchers have slightly different pitches that get miscategorized. Other information included is the batter's name, the location of any baserunners at the beginning of the at-bat, the number of outs, and the result of the at-bat.

What about the location of pitches?

To learn more about a particular at-bat, click on it. On the right side a box will appear that shows the location of each pitch. It will be a gif that shows the sequence of pitches with location and an approximation of the movement illustrated by a faded tail.

What if that is too much information?

Once you reach the pitch chart for a particular pitcher you can filter by game, batter, or situation. If you select a game (by search or dropdown) then every at-bat from that game will be displayed. If you select a batter (by search or dropdown) then every at-bat against that batter will be displayed. If you select a situation then every at-bat satisfying your criteria will be displayed. You can select the number of outs, runners, or inning by checking the appropriate boxes. Note that, for now at least, you cannot combine the situation filters with the game or batter filters.
